# Documentation Directory

This directory stores the configuration and outputs for [Sphinx](https://www.sphinx-doc.org/en/master/index.html), the [autodoc-based](https://www.sphinx-doc.org/en/master/usage/extensions/autodoc.html#module-sphinx.ext.autodoc) documentation generation framework.

Documents are generated in [Markdown](https://www.sphinx-doc.org/en/master/usage/markdown.html) based on [Google-style docstrings](https://google.github.io/styleguide/pyguide.html#38-comments-and-docstrings).
